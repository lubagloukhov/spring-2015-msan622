Data Preprocessing

I preprocessed the data in R. I subset the data to only include observations in the range of 2000 to 2004, inclusive. For each chart, I separately aggregated the data. For the left hand side plot, I calculate sthe proportion of falling into each of the 10 average-ratings buckets. For the upper right hand side plot, I calculated the number of observations falling into each of the aforementioned buckets crossed with each year in the aforementioned range. For the lower right hand side plot, I calculated the number of observations falling into each of the aforementioned buckets crossed with each genre. Additionally, I added an aggration (sum) across all average-ratings backets to assist the plotting of the two right hand side plots when no average-rating bucket is selected. 

library(dplyr)
library(rjson)
 
df <- data.frame(movies)
df$ratingCat <- cut(df$rating, breaks=seq(0,10))
df <- df[df$year>=2000,]
df <- df[df$year<2005,]
 
# LHS
 
pieData <- df %.% 
  dplyr::group_by(ratingCat) %.%
  dplyr::summarise(length(rating)/nrow(df))
names(pieData)<- c("category", "measure")
 
x <- toJSON(unname(split(pieData, 1:nrow(pieData))))
cat(x)
 
# RHS - top 
 
lineData = df[c("year", "ratingCat","title")]
 
lineData2 <- lineData %.% 
  dplyr::group_by(ratingCat, year) %.%
  dplyr::summarise(length(title))
 
names(lineData2) <- c("ratingCat","year", "nObs")
 
lineData3 <- lineData2 %.% 
  dplyr::group_by(year) %.%
  dplyr::summarise(sum(nObs))
lineData3$ratingCat <- "All"
lineData3 <- lineData3[,c(3,1,2)]
 
names(lineData3) <- c("ratingCat","year", "nObs")
 
lineData4 <- rbind(lineData3,lineData2)
 
names(lineData4) <- c("group","category", "measure")
 
x <- toJSON(unname(split(lineData4, 1:nrow(lineData4))))
 
sink("lineData.txt")
cat(x)
sink()
 
# RHS - bottom
 
barData = df[c("Action", "Animation", "Comedy", "Drama", "Documentary", "Romance", 
               "Short", "ratingCat","title")]
head(barData2)
barData2 <- melt(barData, id=c("ratingCat","title"))
 
barData3 <- barData2 %.% 
  dplyr::group_by(ratingCat, variable) %.%
  dplyr::summarise(sum(value))
 
names(barData3) <- c("ratingCat","genre", "nObs")
 
barData4 <- barData3 %.% 
  dplyr::group_by(genre) %.%
  dplyr::summarise(sum(nObs))
barData4$ratingCat <- "All"
barData4 <- barData4[,c(3,1,2)]
 
names(barData4) <- c("ratingCat","genre", "nObs")
 
barData5 <- rbind(barData3,barData4)

write.csv(barData5, "barData.csv", row.names=F)

Functionality Overview

I have implemented C, B and A-level functionality. I have chosen to detail how the number of movies rated and the distribution of genres varies by rating category (0-1,1-2,2-3,...). Though the donut chart is not ideal stylistically, I was not able to implement the single-bar stacked plot that I had prototyped in Tableau. The goruping used to create stacked bars in D3 reaks when there is only one group defined.

User Guide

Click on any portion of the donut to select that category. The bar chart and line chart will filter to only diplay the category selected.
